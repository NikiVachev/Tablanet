# Табланет / Таблич

Базова локална web версия за двама играчи с Node.js, Express и Socket.IO.

## Стартиране

```bash
npm install
npm start
```

Отвори играта на компютъра:

```text
http://localhost:3000
```

## Игра от два прозореца

1. Отвори `http://localhost:3000` в два browser прозореца или два таба.
2. В единия избери `Player 1`.
3. В другия избери `Player 2`.
4. Когато и двамата са свързани, играта започва автоматично.

## Игра от телефон в същата Wi-Fi мрежа

1. Стартирай сървъра на компютъра с `npm start`.
2. Намери локалния IP адрес на компютъра.
   - Linux/macOS: `ip addr` или `ifconfig`
   - Windows: `ipconfig`
3. От телефона отвори адрес от вида:

```text
http://192.168.x.x:3000
```

Замени `192.168.x.x` с IP адреса на компютъра. Компютърът и телефонът трябва да са в една и съща мрежа, а firewall-ът да позволява достъп до порт `3000`.
